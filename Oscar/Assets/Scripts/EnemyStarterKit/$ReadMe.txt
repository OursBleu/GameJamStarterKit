=====================================
INSTRUCTIONS POUR CREER UN PERSONNAGE	
=====================================

Voici les étapes à suivre pour créer un personnage :

1) Créer un dossier <NomDuPersonnage> dans le dossier Scripts
2) Y copier FsmToCopy, ManagerInputToCopy et StateToCopy
3) Renommer ces fichiers en Fsm<NomDuPersonnage>, ManagerInput<NomDuPersonnage> et State<NomPouvoirDuPersonnage>
4) Modifier le contenu de ces fichiers selon les instructions contenues à l'intérieur
5) Créer un gameObject nommé <NomDuPersonnage> avec OBLIGATOIREMENT un collider mais PAS DE RIGIDBODY (il est ajouté automatiquement)
6) Attacher le script Fsm<NomDuPersonnage> à ce prefab (pas les autres, ils sont automatiquement ajoutés)
7) Enregistrer le gameObject en tant que prefab dans le dossier <NomDuPersonnage> créé à l'étape 1
8) C'est fini, le personnage est prêt à bouger, réfléchir et utiliser ses pouvoirs dans un niveau
